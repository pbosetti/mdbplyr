mdbplyr is a dbplyr-inspired interface for MongoDB, implemented as an R package. 
It provides a subset of dplyr verbs that translate into MongoDB aggregation 
stages, allowing users to compose lazy queries against MongoDB collections using 
tidy evaluation.
